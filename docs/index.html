Frequently Asked Questions

How do I create a GameObject?

Extend the GameObject class, implement onLoad() to create your mesh, then add it to a scene:

class MyObject extends GameObject {
        onLoad() {
          this._mesh = new THREE.Mesh(
            new THREE.BoxGeometry(),
            new THREE.MeshStandardMaterial()
          );
        }
      }
How do I switch between scenes?

Use SceneManager to register and load scenes. The manager handles unloading the current scene automatically.

Does the engine support 2D sprites?

Yes — use the SpriteRenderer component or the Sprite asset type for 2D rendering on top of Three.js planes.

What language does the engine use?

BerestaEngine Lite is written in TypeScript and compiled with webpack. Games are written in TypeScript or JavaScript.
