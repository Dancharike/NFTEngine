GameScene abstract

A scene owns and manages a collection of GameObjects. It controls the Three.js scene graph and the full object lifecycle for everything inside it.

import { GameScene } from '@engine/GameScene';

export class LevelOne extends GameScene {
  protected async onLoad(): Promise<void> {
    this.addGameObject(new Player('player'));
    this.addGameObject(new Enemy('enemy'));
  }
  protected onUnload(): void { /* cleanup */ }
}

Abstract Methods

abstract  onLoad(): Promise<void>

Implement to populate your scene. Called once when the scene is loaded. async — you can await asset loading here.

abstract  onUnload(): void

Called after all game objects are destroyed. Clean up non-GameObject resources (timers, subscriptions, etc.).

Object Management

method  addGameObject(gameObject: GameObject): void

Adds an object to the scene. If the scene is already loaded, the object goes through the full awake → load → start pipeline immediately.

method  removeGameObject(id: string): void

Destroys and removes an object by its ID. Triggers onDestroy() on the object.

method  getGameObject(id: string): GameObject | null

Direct lookup by ID.

method  findGameObjectByName(name: string): GameObject | null

Finds the first object whose name matches. Returns null if not found.

method  findByTag(tag: string): GameObject | null

Finds the first object with a matching tag.

method  findAllByTag(tag: string): GameObject[]

Returns all objects with a matching tag.

JS Instance API

A GML-style convenience API for quick prototyping:

MethodDescription
instanceCreate(Class, x, y, z)Creates and adds an object at a position
instanceFind(name)Alias for findGameObjectByName
instanceFindAll(name)All objects with given name
instanceGetAll()All objects in the scene
instanceCount(name)Count of objects with given name
instanceDestroy(object)Destroy a specific object

Broadcasting Messages

method  broadcastMessage(message: IMessage): void

Sends a message to every game object in the scene. Objects receive it via onMessageReceive(message).
